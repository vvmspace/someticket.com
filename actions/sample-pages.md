# Prompt: sample Markdown pages

Generate representative Markdown examples for `someticket.com`.

Requirements:

- The site is statically generated.
- The site language is English.
- Page behavior is controlled by front matter fields such as `page_type` and `entity_type`.
- Use realistic production-oriented examples.
- Include both front matter and body content.

Generate at least these examples:

1. Category hub for `/rock/`
2. City hub for `/new-york/`
3. Artist hub for `/beatles/`
4. Venue hub for a major New York venue
5. Intersection landing for `/new-york/rock/`
6. Event page for `/events/beatles-at-msg-june-12-2026/`

For each file:

- show the proposed file path
- provide full Markdown content
- include metadata fields
- include relationships
- include content block configuration
- include a featured-focus example where relevant

Output must be production-oriented rather than toy examples.
